# Guess-game (python)
This project is a simple number guessing game built python. The program generates a random number and asks the user to guess it.
It provides hints is the guessed number is too high or low , making it an interactive and fun way to practice basic programming concepts.

## features
- Random number generation
- User input for guessing
- Hint system to guide the player (high / too low)
- Easy-to-use command-line interface

## Technologies used
- Python
- Random module

## How to Run the Project
1. Make sure pthon is installed on your system.
2. Download or clone this repository.
3. Open the project folder in your terminal or code editor.
4. Run the program using the following command:
   ...bash
   python number guessing game.py
